# Description
SUBMING is powerfull subdomain scanner for penetration testing

# Instalation & Usage
- git clone https://github.com/irsyadfawwaz/subming.git
- cd subming
- python3 subming.py -d (domain)
- EXP: python3 subming.py -d google.com
